# Manga2Go

Convierte carpetas con imagenes (WebP, JPG, PNG...) a PDF.

## Características

- Conversión por capítulos
- Conversión de una carpeta completa
- Detección automática
- Barra de progreso
- Cancelación
- Reemplazo opcional de PDFs
- Interfaz gráfica
